# Security Policy

## Supported Versions

Only these versions are supported:

---

| Version | Supported          |
| ------- | ------------------ |
| Stable  | :white_check_mark: |
| Nightly | :x:                |
| Bugfix  | :white_check_mark: |
| Dev     | :x:                |

---

## Reporting a Vulnerability

Use [Discussions](https://github.com/mariluski23/fuzzierinter/discussions) to report vulnerabilities.
